# NodeMCU Based Weather Station with RTC DS3231 and DHT11 and Adafruit

NodeMCU Based Weather Station with RTC DS3231 and DHT11 and Adafruit

## Prerequisites
1. NodeMCU V3
2. Arduino IDE
3. TFT LCD 1.77 Inch 160x128
4. DS3231
5. DHT11
6. Push Button
7. 10k Resistors x 2
8. 9v Battery
9. On-Off Button
10. PCB
